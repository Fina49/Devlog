HTML (HyperText Markup Language)

Il s'agit d'un langage qui formalise l'écriture d'un document web à l’aide de balises.
Une page HTML est un fichier texte contenant des balises permettant de structurer le contenu de la page
C’est donc un document texte avec une extension de fichier particulière ‘.html’ qui signale que
le texte à l’intérieur du fichier est bien à comprendre comme du code HTML.

Structure générale d'un document HTML :

La première ligne du document est indispensable car elle indique qu'il s'agit bien d'une page web HTML. En HTML5 on écrit simplement <!DOCTYPE html>.
Ensuite, la balise <html> </html> englobera l’ensemble des autres balises de la page. Elle contient deux paires de balises:
La balise <head> </head> délimitant l’entête du document et contenant diverses informations sur le document lui-même (métadonnées).
Son contenu n’est pas visible dans la page interprétée par le navigateur. A noter :

  • l’encodage du texte meta charset="utf-8"/ pour éviter les problèmes d’affichage des caractères spéciaux
  • La balise <title> Titre de la page <title> qui contient le titre (assez court) qui s’affiche dans l’onglet de lafenêtre du navigateur.
  • La balise <body> </body> délimitant le corps du document. Cette portion contient la partie visible du document HTML. C’est ici qu’on trouvera les autres balises (de différents types) structurant le document.
